Website Evaluation: Baltimore City Schools

Website Information

URL: https://www.baltimorecityschools.org

Name: Baltimore City Public Schools

Target Audience: Students, parents, teachers, and staff within the Baltimore school district.

Website Organization

The website has a clear navigation bar at the top with links to main section like "Schools," "Parents," "Students," and "Staff." The homepage provides quick access to important updates, announcements, and a search bar for easier navigation.

CRAP Design Principles

The website uses the Contrast well with a bold dark-blue navigationbar on ta white background It also repeats the same branding and color style on every page.

Accessibility Audit

Audit Score: The audit score shows 61. accessibility tool reported 5 errors, including a missing alt text for one image and one sufficient size and space.

Effectiveness

The website makes it easy for users to quickly and accruately find school resources and district imformation with its simple layout.

Efficiency

Most tasks, like finding school infromation or accessing forms ,are easy to do because of the well-organized menu.

Recommendation for Improvement

To improve accessibility, the website should add proper alt text to all images.
